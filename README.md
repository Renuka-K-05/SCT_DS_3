## Task 3: Customer Purchase Prediction using Decision Tree Classifier

### Objective

Build a Decision Tree Classifier to predict whether a customer will subscribe to a term deposit based on demographic and behavioral data from a bank marketing campaign.

### Dataset

Bank Marketing Dataset (UCI Machine Learning Repository)

### Tools and Libraries

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

### Project Workflow

#### 1. Data Preprocessing

* Loaded and explored the dataset
* Checked for missing values and duplicates
* Performed feature engineering and data cleaning
* Prepared the dataset for machine learning

#### 2. Exploratory Data Analysis (EDA)

* Analyzed customer demographics and behavioral attributes
* Examined relationships between features and the target variable
* Visualized patterns and trends using charts and graphs

#### 3. Feature Selection and Encoding

* Selected relevant features for model training
* Converted categorical variables into numerical format suitable for machine learning algorithms

#### 4. Model Building

* Split the dataset into training and testing sets
* Built a Decision Tree Classifier using Scikit-learn
* Trained the model on customer demographic and behavioral data

#### 5. Model Evaluation

* Evaluated model performance using classification metrics
* Generated a confusion matrix
* Visualized the Decision Tree structure for interpretation

### Key Outcome

The Decision Tree Classifier was successfully developed to predict customer subscription behavior. The project demonstrates the complete machine learning workflow, including data preprocessing, exploratory analysis, model training, evaluation, and interpretation.
